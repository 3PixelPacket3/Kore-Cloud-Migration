    How to Use

    Overview

    Nebula Layout Forge is a fully offline editor for creating structured, styled HTML content. It is ideal for documentation, FAQs, changelogs, landing sections, and more.

    Basic Workflow

    1. Go to the Editor & Preview tab and type your content in the main editor area.
    2. Use the toolbar and left rail to apply styles: headings, lists, boxes, badges, tables, timelines, and spacers.
    3. Use the preview device controls (Desktop / Tablet / Mobile) to see how your layout behaves at different widths.
    4. Switch to the HTML Output tab or use the Copy HTML button in the editor header to copy the generated HTML.

    Presets & Templates

    • Use Quick Presets from the editor side rail or the full Templates tab to load starter layouts. When a preset or template is loaded, you are returned to the editor view.
    • Save your favorite configurations as My Templates (up to five), then reload them whenever you need.

    Backups

    • The app autosaves your work to localStorage as you type.
    • Use Backup & Restore to export a JSON file containing the full app state (content, settings, templates) or to restore from a previous backup.

    Using the Generated HTML

    The generated HTML is designed to be portable. You can paste it into:

    • Static websites or partials.
    • CMS editors that support raw HTML.
    • Documentation systems or internal tools.

    Copy the relevant CSS utilities for boxes, tables, timelines and badges into your project’s stylesheet to preserve the look and feel.

    Advanced Controls & Tips

    Layout Components

    • Use Accent Sections to frame important groups of content with colored backgrounds and borders.
    • Two Column snippets are ideal for side-by-side comparisons or features.
    • Timeline snippets visualize sequences such as roadmaps or incident reviews.

    Tables & KPIs

    • The KPI Board preset table is perfect for status dashboards and quick health checks across projects.
    • The Chart-like table style emphasizes header rows and key columns for a pseudo-visual look without actual charts.
    • The Matrix table is tuned for feature vs plan comparisons.

    Typography & Spacing

    • Use A− / A+ to make specific text smaller or larger relative to the base size.
    • The line-spacing control in the toolbar lets you apply compact/relaxed styles to selected text blocks.
    • The font selector in the toolbar can apply a Sans, Serif, Mono, or Display font to a specific selection.

    Inline Help

    Most toolbar buttons include titles that show brief descriptions on hover. For complex components like KPI tables, timelines, and grids, start by inserting the component, then overwrite the placeholder content.
