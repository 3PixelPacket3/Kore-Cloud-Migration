## Documents Library – Deep Guide

The Documents Library stores references, files, and supporting materials.

### Supported File Types
- PDF
- TXT / MD
- Images (PNG, JPG, JPEG)

Documents are indexed locally and can be categorized and searched.

### Use Cases
- Training manuals
- SOPs
- Reference PDFs
- Screenshots and diagrams

### Tips
- Use clear file names
- Group related documents by category
- Pair documents with Info Hub guides
