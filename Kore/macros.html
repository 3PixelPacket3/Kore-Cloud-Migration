LOD Data Cleanser Excel VBA

Automatically strips hyphens from ZIP codes and formats columns as text to prevent leading zeros from dropping.

Sub CleanLODFile()
    Dim ws As Worksheet
    Set ws = ActiveSheet
    
    ' Format Col L (ZIP Code) as Text
    ws.Columns("L").NumberFormat = "@"
    
    ' Remove Hyphens from Col L
    ws.Columns("L").Replace What:="-", Replacement:="", LookAt:=xlPart
    
    ' Ensure Col B (Customer #) is Text
    ws.Columns("B").NumberFormat = "@"
    
    MsgBox "LOD Cleansing Complete", vbInformation
End Sub

Azure Table Storage Ingest Python

Basic Python script using the Azure SDK to push a JSON payload into the cloud database.

from azure.data.tables import TableServiceClient
import json
import os

def push_to_azure(data_dict, table_name="KoreLog"):
    conn_str = os.environ.get("KORE_DB_CONNECTION")
    service_client = TableServiceClient.from_connection_string(conn_str=conn_str)
    
    table_client = service_client.get_table_client(table_name=table_name)
    
    entity = {
        "PartitionKey": "AutoIngest",
        "RowKey": str(data_dict.get("id", "001")),
        "Payload": json.dumps(data_dict)
    }
    
    table_client.upsert_entity(entity=entity)
    print("Payload pushed successfully.")

Fetch Entra ID Profile JavaScript

The standard asynchronous function used across Kore to extract user identity from the Azure proxy.

async function fetchSecureIdentity() {
    try {
        const response = await fetch('/.auth/me');
        const data = await response.json();
        
        if (data.clientPrincipal) {
            const userId = data.clientPrincipal.userId;
            const userName = data.clientPrincipal.userDetails;
            console.log(`Authenticated as: ${userName}`);
            return { id: userId, name: userName };
        }
    } catch (error) {
        console.warn("Authentication endpoint unreachable. Running locally.");
        return null;
    }
}

Automated Key Sequence Power Automate / VBS

A standard send-keys loop for repetitive terminal entry. Ensure window focus before execution.

Set WshShell = WScript.CreateObject("WScript.Shell")
WScript.Sleep 2000 ' Wait 2 seconds for focus

' Loop through standard iSeries entry
For i = 1 To 10
    WshShell.SendKeys "1NAPR"
    WshShell.SendKeys "{TAB}"
    WshShell.SendKeys "O"
    WshShell.SendKeys "{ENTER}"
    WScript.Sleep 500 ' Delay to allow terminal response
Next

MsgBox "Sequence execution finished."
